1 Quem somos

O ProduzAI é uma plataforma SaaS de controle de produção para empresas do setor têxtil e de confecção, desenvolvida e operada por Lívia Cordeiro Ribeiro (MEI/CNPJ em processo de abertura), com sede no Brasil.

Esta política se aplica a todos os usuários da plataforma, incluindo empresas contratantes e seus funcionários que utilizam o sistema.

2 Dados que coletamos

Coletamos apenas os dados necessários para o funcionamento da plataforma:

  • Dados de conta: nome, endereço de e-mail e senha (armazenada com hash seguro)
  • Dados da empresa: nome fantasia, logo (opcional), metas de produção
  • Dados operacionais: registros de produção, nome dos operadores, quantidade de peças, defeitos, produtos e ordens de produção
  • Imagens de formulários: fotos tiradas pelo usuário para processamento via IA — as imagens são enviadas à API da Anthropic e não são armazenadas permanentemente
  • Dados técnicos: endereço IP, tipo de navegador e logs de acesso para fins de segurança
Não coletamos dados sensíveis como CPF, RG, dados bancários, informações de saúde ou biometria dos funcionários.

3 Como usamos os dados

Seus dados são usados exclusivamente para:

  • Autenticação e controle de acesso à plataforma
  • Armazenamento e exibição dos registros de produção da sua empresa
  • Geração de dashboards, relatórios e análises de desempenho
  • Processamento de imagens de formulários via Inteligência Artificial
  • Envio de comunicações transacionais (confirmação de conta, alertas de trial)
  • Suporte técnico quando solicitado

Não usamos seus dados para publicidade, não vendemos dados a terceiros e não realizamos perfilamento para fins comerciais.

4 Processamento por Inteligência Artificial

O ProduzAI utiliza a API Claude Vision da Anthropic para extrair dados de formulários físicos fotografados pelos usuários.

Como funciona: a imagem é enviada criptografada à API da Anthropic, processada para extração dos dados e o resultado (texto estruturado) é retornado ao app. A imagem original não é armazenada em nossos servidores.
  • As imagens são transmitidas via HTTPS com criptografia TLS
  • A Anthropic processa os dados conforme sua própria Política de Privacidade
  • Não utilizamos os dados para treinar modelos de IA
  • O usuário pode optar pelo modo de digitação manual, sem envio de imagens

5 Compartilhamento de dados

Compartilhamos dados apenas com os seguintes fornecedores essenciais para operação da plataforma:

  • Supabase (supabase.com) — banco de dados e autenticação, servidores na região South America (São Paulo)
  • Anthropic (anthropic.com) — processamento de imagens via IA
  • Vercel (vercel.com) — hospedagem da aplicação
  • Stripe (stripe.com) — processamento de pagamentos (apenas para dados de cobrança)

Todos os fornecedores são contratualmente obrigados a proteger seus dados e utilizá-los apenas para prestação do serviço contratado.

Não compartilhamos dados com anunciantes, parceiros comerciais ou outras empresas não listadas acima.

6 Armazenamento e segurança

Adotamos as seguintes medidas para proteger seus dados:

  • Criptografia Dados em repouso e em trânsito criptografados via TLS/HTTPS
  • RLS Row Level Security ativo — cada empresa acessa apenas seus próprios dados
  • Auth Autenticação segura via Supabase Auth com suporte a OAuth (Google)
  • Isolamento Dados de empresas diferentes são completamente isolados no banco
  • Backup Backups automáticos realizados pelo Supabase

Em caso de incidente de segurança que afete seus dados, notificaremos os usuários impactados em até 72 horas, conforme exigido pela LGPD.

7 Cookies e rastreamento

O ProduzAI utiliza apenas cookies essenciais para funcionamento da plataforma:

  • Sessão de autenticação: necessário para manter o usuário logado
  • Preferências locais: tema (claro/escuro) e configurações salvas no dispositivo

Não utilizamos cookies de rastreamento, pixels de publicidade ou ferramentas de analytics de terceiros.

8 Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem os seguintes direitos:

  • Acesso: solicitar cópia de todos os dados pessoais que armazenamos sobre você
  • Correção: solicitar correção de dados incompletos, inexatos ou desatualizados
  • Exclusão: solicitar a exclusão de seus dados pessoais (conta e registros)
  • Portabilidade: receber seus dados em formato estruturado (CSV/JSON)
  • Revogação: revogar o consentimento para tratamento de dados a qualquer momento
  • Oposição: opor-se ao tratamento de dados em determinadas circunstâncias
Para exercer qualquer um desses direitos, entre em contato pelo e-mail suporte@produzai.com.br. Responderemos em até 15 dias úteis.

9 Retenção dos dados

  • Dados de conta ativa: mantidos enquanto a conta estiver ativa
  • Após cancelamento: dados mantidos por 30 dias para possível reativação, depois excluídos permanentemente
  • Trial expirado sem assinatura: dados excluídos após 30 dias do fim do trial
  • Logs de segurança: mantidos por até 90 dias para fins de auditoria

Você pode solicitar a exclusão imediata dos seus dados a qualquer momento, independente do prazo acima.

10 Alterações nesta política

Podemos atualizar esta política periodicamente. Quando houver alterações relevantes, notificaremos os usuários por e-mail ou por aviso dentro da plataforma com pelo menos 15 dias de antecedência.

O uso continuado da plataforma após as alterações implica na aceitação da nova política.
